Stylus only

Instructions

  • Select bead type on the left
  • Draw on the grid with mouse or touch (if "stylus only" is checked, only pressure-enabled touches are registered)
  • "Save" button opens the grid as an SVG image so it can be manually saved in the browser
  • "Clear" button resets all cells in the grid
  • Use directional arrows to move all the beads on the grid (to re-center)
